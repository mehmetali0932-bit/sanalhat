# SanalHat - Sanal Numara Kiralama Platformu

Bu proje sanal numara ve SMS dogrulama hizmeti saglayan bir demo uygulamasidir.

## Ozellikler
- Kayit ve Giris (bcrypt)
- Demo Bakiye (100 kredi)
- Numara Satinlama
- REST API
- Admin Paneli

## Kurulum

```bash
npm install
npm start
```

http://localhost:3000
